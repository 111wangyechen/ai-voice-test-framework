# AI 语音测试工具 — 使用说明

## 快速开始

双击 `ai-voice-test.exe`，等待几秒，浏览器会**自动打开**测试界面。
无需手动启动前后端，也无需输入端口地址。

如果浏览器没有自动打开，手动访问：http://localhost:8000

关闭：直接关掉弹出的黑色命令行窗口即可停止服务。

---

## 两种使用模式

### Mock 模式（无设备，开箱即用）
不连接任何设备，直接点「开始测试」即可看到模拟的日志和性能数据，
用于演示和熟悉界面。

### 真实设备模式
需要系统已安装 **adb** 并加入 PATH（命令行能执行 `adb version`）。

1. 在界面中填写设备 IP（如 `10.7.187.34:5555`），点连接
2. 开始测试，查看真实设备的日志解析和性能监控
3. 生成报告

> 验证 adb 是否就绪：打开命令行执行 `adb version`，能输出版本号即可。
> 若未安装，请安装 Android Platform Tools 并将其目录加入系统 PATH。

---

## 端口说明

默认使用 **8000** 端口。如果该端口被占用，可在启动前设置环境变量：

```
set PORT=8080
ai-voice-test.exe
```

---

## 重新打包（开发者）

修改代码后，在项目根目录执行：

```bash
bash build_exe.sh
```

会自动重新构建前端、打包后端，产物输出到 `release/ai-voice-test.exe`。
